<h3>职位列表</h3>

首页请求职位列表的接口：

（请求）2017.07.15
```
type: get
url: http://121.41.66.68/job/getJobList
data:
说明：
1. 缺少和没有的字段，默认设为在该字段无限制
2. 数字全部转为字符串传递；
3. 有todo的为修改项（相比初始版本）

{
    // type: array
    // limit: 5 elements && maxlength = 6 per element && 无敏感词（后端校验）
    // 搜索关键字，搜索框输入内容
    // 例如搜索框输入内容是："a b"，那么自动拼接为数组["a", "b"]
    // 本项内容会自动匹配【关键词】、【area】
    keywords: ["高薪", "弹性工作制"],

    // type: string
    // limit: length < 10
    // 区域
    area: "上海市浦东区",

    // type: string
    // limit:  length < 3 && value > -1 && 整数
    // 最小年限，0表示不限制
    // todo 有修改
    minYear: "0",

    // type: string
    // limit: length === 1&& value > -1 && 整数
    // 学历：0: 不要求，1: 大专，2: 本科，3: 硕士，4: 博士
    academic: "",

    // type: string
    // limit: length === 1 && value > -1 && 整数
    // 融资程度：0:无限制，1:不需要融资，2:未融资，3:天使轮，4:A轮，5:B轮，6:C轮，7:D轮及以上，8:上市
    // todo 有修改
    financing: "0",

    // todo 删除本条
    // industry: "",

    // type: string
    // limit: value === 0 || value === 1
    // 排序：0，正序，1，倒序，默认0
    // todo 暂不生效
    sorting: "0",

    // type: string
    // limit: value > -1 && value < 100k && length < 6
    // 最低工资要求（月薪），0表示无限制
    // todo 有修改
    minWage: "0",

    // type: string
    // limit: value > -1 && value < 100k && length < 6
    // 最高工资要求（月薪），0表示无限制
    // todo 有修改
    maxWage: "99999",

    // type: string
    // limit: > 0 && value < 10000 && 整数
    // 第几页（分页查找），第一页是1
    page: "1"
}
```
